# Hybrid_Model_for_Detecting_Alcoholism-_from_EEG_Signals_using_DL_ML
Hybrid Model for Detecting Alcoholism from EEG Signals 🧠🤖Este proyecto presenta un modelo híbrido de última generación para la detección automática del alcoholismo, combinando la potencia de extracción de características del Deep Learning con la robustez clasificatoria del Machine Learning.📊 Resultados DestacadosEl modelo final, basado en la arquitectura ViT + SVM, alcanzó un desempeño excepcional en el conjunto de datos de prueba:Exactitud (Accuracy): 98.38%.Sensibilidad (Recall): 100% — Garantiza la detección de todos los casos positivos sin omisiones.Precisión: 96.91%.AUC-ROC: 0.9996.🛠️ MetodologíaEl flujo de trabajo se divide en cuatro fases críticas:Adquisición: Uso de señales EEG (Dataset de UCI/Kaggle).Preprocesamiento: Generación de escalogramas mediante la Transformada Wavelet Continua (CWT), normalizados y redimensionados a 224x224 px.Extracción de Características: Evaluación de arquitecturas como ViT, CNN, MobileNetV2 y MLPMixer.Clasificación: Implementación de SVM, Random Forest y k-NN con optimización de hiperparámetros.📂 Estructura del Proyectonotebooks/: Proceso completo de entrenamiento y evaluación.src/: Scripts con las definiciones de los modelos híbridos.models/: Pesos del mejor modelo guardado (ViT + SVM).docs/: Artículo científico y presentación de sustentación.🚀 TecnologíasLenguaje: Python.Frameworks: TensorFlow, Keras y Scikit-Learn.Análisis de datos: Pandas, NumPy, Matplotlib y Seaborn.
